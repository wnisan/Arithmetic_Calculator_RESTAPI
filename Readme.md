Калькулятор арифметических выражений, реализованный с использованием TDD (Test-Driven Development) подхода.

Возможности:

- Преобразование выражений из инфиксной нотации в обратную польскую нотацию (RPN)
- Вычисление выражений в RPN формате
- Поддержка операторов: `+`, `-`, `*`, `/`
- Обработка скобок `()` 
- Валидация выражений с кастомными ошибками
- Полное покрытие тестами
